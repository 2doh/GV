커스텀 훅 정리
usePlaceholder : input 태그 focus시 placeholder 제거

<AuthInput>비밀번호를 입력해 주세요</AuthInput>

AuthInput.tsx
```
interface AuthData {
  children: string;
}

const { placeholder, handleFocus, handleBlur } = usePlaceholder(children);

placeholder={placeholder}
onFocus={() => handleFocus()}
onBlur={() => handleBlur()}
```
